Ransomware on S3 - Simulation and Detection

This project creates a safe space to practice spotting ransomware attacks in AWS storage (S3 buckets). It simulates an attack by setting up a fake environment and then throwing a virtual ransomware script at it. By examining logs and other data (costs, additional logs), you can see how well you can detect the ransomware's trail, including if it stole information or disabled security measures. Think of it like a fire drill for your AWS storage!
